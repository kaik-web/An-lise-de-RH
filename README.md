Dashboard interativo desenvolvido no Power BI para análise de indicadores de Recursos Humanos, com foco em turnover, distribuição de equipes e mapeamento de evasão por departamento.
 
Objetivo:
Simular o trabalho de um analista de dados no contexto de RH, transformando dados brutos de funcionários em insights acionáveis para a gestão de pessoas.

Indicadores Apresentados:
KPIDescriçãoTotal de FuncionáriosHeadcount geral da empresaFuncionários AtivosColaboradores com vínculo ativoFuncionários DesligadosColaboradores que deixaram a empresaTaxa de TurnoverPercentual de rotatividade da equipe

Visuais do Dashboard:

Funcionários por Departamento — distribuição do headcount por área
Colaboradores por Nível de Senioridade — pirâmide de maturidade da equipe
Distribuição por Gênero (Ativos) — gráfico de rosca com percentuais
Mapeamento de Evasão — ativos vs. desligados por departamento

Ferramentas Utilizadas:
Power BI Desktop — modelagem e visualização
SQL Server — armazenamento e consulta dos dados
DAX — criação das medidas e KPIs

Sobre os Dados:
Os dados utilizados neste projeto são 100% fictícios, criados manualmente para fins de estudo e demonstração. Não representam nenhuma empresa real.
